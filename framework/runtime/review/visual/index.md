# Visual Review Bank

## Purpose
This bank defines PDF-only visual reviewers for presentation review.

These reviewers may see only:
- PDF
- review brief
- outline

These reviewers may not see:
- code
- HTML
- CSS
- package files
- runtime files
- implementation details

## Reviewers

### theme-reviewer
Focus:
- theme coherence
- visual family
- overall visual language

Use when:
- checking whether the deck feels like one artifact
- checking whether surfaces, colors, and treatments belong together

### layout-reviewer
Focus:
- composition
- alignment
- balance
- structural layout quality

Use when:
- checking whether slides are well-built spatially
- checking whether balance, symmetry, and asymmetry feel intentional

### typography-reviewer
Focus:
- readability
- hierarchy
- type discipline

Use when:
- checking whether text can be read and understood comfortably

### density-reviewer
Focus:
- crowding
- breathing room
- visual pacing

Use when:
- checking whether slides are overloaded or under-built

### consistency-reviewer
Focus:
- repeated patterns
- slide-type drift
- component consistency

Use when:
- checking whether similar slides behave similarly

### media-reviewer
Focus:
- screenshots
- charts
- tables
- image treatment

Use when:
- checking whether visual evidence is handled cleanly

### polish-reviewer
Focus:
- finish
- refinement
- subtle defects

Use when:
- checking whether the deck feels complete and presentation-ready
