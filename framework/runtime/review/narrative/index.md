# Narrative Review Bank

## Purpose
This bank defines artifact-only narrative reviewers for presentation review.

These reviewers may see only:
- PDF
- brief
- outline

These reviewers may not see:
- code
- HTML
- CSS
- package files
- runtime files
- implementation details

## Reviewers

### message-reviewer
Focus:
- core message
- headline thesis
- main takeaway

### audience-reviewer
Focus:
- audience fit
- framing assumptions
- what the intended reader can understand

### structure-reviewer
Focus:
- narrative sequence
- section order
- slide-to-slide logic

### clarity-reviewer
Focus:
- copy clarity
- jargon pressure
- cognitive load

### evidence-reviewer
Focus:
- claim support
- proof sufficiency
- evidence-to-claim linkage

### objection-reviewer
Focus:
- skeptical gaps
- unresolved questions
- missing counterpoints

### close-reviewer
Focus:
- ending strength
- decision clarity
- ask and takeaway
