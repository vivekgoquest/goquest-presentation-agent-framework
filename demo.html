FrameworkPitch Framework v1.0

Slide Format16:9 landscape, 960 x 540pt PDF pages
PDF ExportPlaywright screenshots at 3x DPI + pdf-lib assembly

TypographyInter font, responsive clamp() sizing
Grids.g2 .g3 .g4
The canvas layer is sacrosanct — slide dimensions, aspect ratio, and grid system never change. The theme inherits that structure, and content adds deck-specific sections without redefining framework primitives.

Two-Column Layout

The .g2 grid creates a balanced two-column layout.

All grids collapse to single-column on mobile (below 768px).
